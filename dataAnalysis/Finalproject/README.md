# BikeShare Analysis Project

## Overview

This project analyzes bike sharing data using Python and Pandas.

Dataset: Ford GoBike trip data

Goal:

- Clean dataset
- Create new features
- Perform exploratory data analysis
- Prepare data for machine learning
